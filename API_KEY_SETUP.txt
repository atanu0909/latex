==================================

IMPORTANT: Keep your API key secure and never commit it to Git!

METHOD 1: Using thbe Setup Script (Recommended)
-----------------------------------------------
Run this command in the terminal:
    ./setup_api_key.sh

This will prompt you to enter your API key and create a .env file.


METHOD 2: Manual Setup
----------------------
1. Get your API key from: https://makersuite.google.com/app/apikey

2. Create a file named `.env` in the project root

3. Add this line to the .env file:
   GEMINI_API_KEY=your_actual_api_key_here

4. Save the file


METHOD 3: Set Environment Variable Directly
--------------------------------------------
Linux/Mac:
    export GEMINI_API_KEY=your_actual_api_key_here

Windows:
    set GEMINI_API_KEY=your_actual_api_key_here


RUNNING THE APPLICATION
------------------------
After setting up the API key:

Option 1 - Use the start script:
    ./start.sh

Option 2 - Run manually:
    export GEMINI_API_KEY=your_key_here  # (if not using .env)
    python app.py

The app will be available at: http://localhost:5000

TROUBLESHOOTING
---------------
If you see "API key not configured":
- Make sure you've set the environment variable OR created .env file
- Check that the .env file contains: GEMINI_API_KEY=your_key_here
- Try restarting the application after setting the key
